# TESTNET

## Prerequisites

[Rust](https://www.rust-lang.org/tools/install), [Golang](https://golang.org/doc/install) >= 1.13

## Build

`$ make install`

## Run
* Daemon: `$ ./nodef`
* CLI: `$ ./clif`

## Test

`$ make test`
